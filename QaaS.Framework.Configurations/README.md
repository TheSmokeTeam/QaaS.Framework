# QaaS.Framework.Configurations

A package for shared configuration loading logics/objects in the QaaS framework.

## Adding Custom Validation Attributes

If you want to add further validation to your configuration you can do so by creating and using
custom `Validation Attributes` which are classes that inherit from `ValidationAttribute` and override its `IsValid` method.

To see the custom validation attributes in this package click [here](CustomValidationAttributes).
