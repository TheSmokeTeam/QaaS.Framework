# QaaS.Framework.SDK

This is a nuget package containing all core elements of the QaaS Framework, in order to write
any pluggable hooks that combine with QaaS this package must be used.

[TOC]

## Supported Hooks

### IGenerator

Extends `IHook`

Represents a data generator

### IAssertion

Extends `IHook`

Represents a test assertion

## Important Core Objects

### DataSource

Represents a loaded data from certain source, could be generated by nothing, or by certain external source.
Contains all relevant data to run a generator and get its generated data with a name tag.

### SessionData

Contains all relevant data of a session that was performed.

### SerializedSessionData

Contains all relevant data of a session that was performed after serializing all of the body of the communication data
items according to their serialization types.

### Context

Contains all QaaS objects that you should have access to from anywhere in your test project.
